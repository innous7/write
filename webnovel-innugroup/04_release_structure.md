# 04. 연재 운영 구조

## 기본 운영
- 기준 기간: 2010년~2026년 (총 17년)
- 연간 회차: 주인공 1인당 연 10화
- 연간 총화수: 50화 (5인 x 10화)
- 방식: 같은 주차 사건을 5명 시점으로 각각 전개

## 권장 요일 편성
- 월: 강서준(물류)
- 화: 한도희(엔터)
- 수: 윤재훈(투자)
- 목: 문하린(푸드)
- 금: 오민석(그룹)

## 주차별 공통 설계
- `공통 사건 1개`: 같은 사건을 5개 시점으로 분해
- `개별 사건 1개`: 각 주인공의 독립 플롯 전진
- `떡밥 1개`: 다음 주차 공통 사건으로 연결

## 독자 경험 설계
- 월~목: 정보 비대칭 축적 (독자가 퍼즐 조립)
- 금: 민석 화에서 권력 정보 공개로 한 주 마무리
- 주말: 작가 노트/인물 카드 보강(선택)

## 분량 가이드
- 1화 권장: 5,000~6,000자
- 주간 총량: 25,000~30,000자
- 연간 총량: 250,000~300,000자 (50화 기준)
- 전체 총량(2010~2026): 4,250,000~5,100,000자 (850화 기준)

## 장기 편성 수치
- 전체 회차 공식: `17년 x 10화 x 5인 = 850화`
- 인물별 누적 회차: 170화
- 운영 권장: 연간 10주 메인 블록 + 비정기 휴재/특별편

## 집필 운영 규칙
- 집필 순서 고정: 강서준 -> 한도희 -> 윤재훈 -> 문하린 -> 오민석
- 같은 만남 장면의 대사는 5인 화에서 동일 문장으로 유지
- 새 화 집필 전 `07_writing_principles.md` 체크리스트를 필수 수행
