# 이뉴그룹 멀티 주인공 웹소설 자료실

## 프로젝트 콘셉트
- 장르: 기업 성장 드라마, 권력 투쟁, 군상극
- 구조: 5인 주인공 병렬 서사 (동일 세계관, 각기 다른 시점)
- 배경: 2010년 이뉴컴퍼니(커머스 스타트업) -> 2026년 주식회사 이뉴그룹(대기업화)
- 최종 갈등: 5인이 축적한 지분 연합으로 회장을 몰아낸 뒤, 차기 회장 경쟁에 돌입

## 파일 구성
- `01_worldbuilding.md`: 세계관/기업 구조/핵심 갈등 축
- `02_characters.md`: 캐릭터 바이블(고정 성격/관계/핵심 서사)
- `characters-by-year/`: 연도별 등장인물 상태 시트 (2010~2026)
- `03_timeline_2010_2026.md`: 17년 연표
- `04_release_structure.md`: 주간 5화 연재 운영안
- `05_season1_episode_blueprint.md`: 시즌1(입사 초기) 회차 청사진
- `06_org_structure_2010.md`: 2010년 조직도/부서별 팀장·팀원(총 50명)
- `07_writing_principles.md`: 집필 필수 원칙(연결성/대사 일치/사전 검토 체크리스트)
- `08_story_update_log.md`: 연재 중 설정 추가/수정 이력 로그
- `episodes/`: 실제 집필 원고(연도/주차/인물별 회차 파일)

## 설정 메모
- 요청 반영 기준: 주인공 성별 구성: 남성 3명(강서준·윤재훈·오민석), 여성 2명(한도희·문하린).
